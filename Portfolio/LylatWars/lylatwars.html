Lyat Wars PS4 Remake

With this being a project done on the PlayStation 4, it is under an NDA. This means the source code can be or will be released.

This project involved making a 3D game using the PlaySation 4 dev kits; these are powerful kits which was given to me through Northumbria University.

The first thing which had to be done in the project before making the game was to make the backend of the game including how objects were to be displayed on screen, the camera, loading shaders to the GPU, collisions and much more.

With Lyat Wars (or better known as Star Fox 64 as its better known as) being a game passed down to me when younger, it was a game I adored. Therefore I though that recreating as much as I could on the PlayStation 4 dev kits felt appropriate.
